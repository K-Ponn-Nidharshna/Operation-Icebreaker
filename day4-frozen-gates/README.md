# Day 4 — Frozen Gates: Password Strength Checker
**Operation IceBreaker | K. Ponn Nidharshna**

## What This Tool Does
Scores a password based on length, character variety, 
and checks it against a common password list, then 
gives actionable feedback.

## What Makes a Password Strong?
- Minimum 12+ characters
- Mix of uppercase and lowercase letters
- Contains numbers
- Contains special characters (!@#$...)
- Not a commonly used password 

## Why Weak Passwords Fail
Attackers use brute force and dictionary attacks to 
crack passwords. Short or common passwords can be 
cracked in seconds. A strong password can take 
billions of years to crack.

## How to Run
python password_strength_checker.py

- Python 3
