# Frostwall Toolkit — Operation IceBreaker
*Days 8–9 | K. Ponn Nidharshna* 

## What is Frostwall?
A menu-driven command-line security toolkit combining 
three defensive tools built during Operation IceBreaker.

## Tools Included
1. **File Integrity Checker** — Detects file tampering 
   using SHA-256 hashing
2. **Password Strength Checker** — Scores passwords 
   and gives feedback
3. **Log Analyzer** — Detects brute force attacks 
   from server logs

## How to Run
pip install pandas
python frostwall.py

## Technologies Used
- Python 3
- hashlib (SHA-256 hashing)
- re (regular expressions)
- pandas (log data analysis)
- json (hash storage)
