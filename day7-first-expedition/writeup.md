TryHackMe — Intro to Cyber Security Path

## 1. Offensive Security
Offensive security means thinking like an attacker 
and legally finding vulnerabilities before bad guys 
do. Key techniques include reconnaissance, scanning, 
exploitation, privilege escalation, and reporting.

## 2. Defensive Security
Defensive security focuses on protecting systems and 
detecting attacks. Key techniques include threat 
monitoring, log analysis, incident response, digital 
forensics, and threat intelligence.

## 3. Careers in Cybersecurity
| Role | What they do |
|---|---|
| SOC Analyst | Monitor alerts and threats daily |
| Penetration Tester | Hack systems legally |
| Incident Responder | Handle active attacks |
| Malware Analyst | Study viruses and malicious code |
| Forensics Analyst | Investigate breaches |

## 4. Inside a Computer System
- Files and Permissions — every file has an owner 
  and access level
- Processes — every running program is a process; 
  malware hides as one
- Users and Accounts — attackers try to reach 
  admin level
- Memory — attackers inject malicious code into RAM
- Network Connections — suspicious ones signal malware
